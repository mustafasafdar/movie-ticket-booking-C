# Movie Ticket Booking System (C)
A console-based seat booking system built in C as a PBL (Problem-Based Learning) project for Computer Programming (CPE-121T) at the Department of Computer Engineering, Bahauddin Zakariya University (BZU), Multan.

## Features
- Displays a 5x5 seat grid (O = Available, X = Booked)
- Takes user input for row and column selection
- Uses pointer-based logic to update seat status
- Validates seat range and checks for already-booked seats

## Concepts Used
- 2D Arrays
- Pointers
- Functions
- User Input Handling (scanf)

## How to Run
Compile and run using any C compiler:
gcc movie_ticket_booking.c -o movie_ticket_booking./movie_ticket_booking
